Agile Toolkit Tutorial

Your path of becoming trained in a way of Agile Toolkit consists of 3 trials. All of the trials are simple and highly interractive. You do not need to finish first trial to start second, you can learn them in parallel.

First Trial — User Interface Development

Here you will learn how to quickly create a web form, menus, pages, grids and other common Web elements without writing any HTML or JS code. After completing this tutorial you will know how to handle Views.

Start with User Interface Tutorial

Second Trial — Functional Development

This path leads to the knowledge of developing business logic on Models and MySQL. You will also learn how to develop multi-table models, referenced models, conditionals, validation, calculated fields, model inheritance and use of SQL stored procedures.

Start with User Interface Tutorial

Third Trial — Extending

Agile Toolkit is a PHP-based Web Software Development Environment. This tutorial is designed to be very visual and will give you maximum visual results from the very beginning. It is also focused around the real-life requirements of typical web applications and by following it you will be able to implement your Web Software in the shortest time possible.

By following guidelines of Agile Toolkit you will also ensure longevity of your software. The way of Agile Toolkit is the way of a flexible an scalable software in terms of performance and feature enhancements

Front-end and Back-end

Agile Toolkit offers you an integrated environment where you as developer would need to look after front-end and the back-end. The development models for those two areas are very distinctive, even the code looks very different. Agile Toolkit binds both parts together in a seamless integrated way.
