Translating www.agiletoolkit.org

Translations of documentation on agiletoolkit.org are user-contributed. We are willing to provide documentation in many new languages, but we need your help to translate them.

To translate you need to fork Github repository of agiletoolkit.org and populate "locale" directory with translated copy of templates. You can always install agiletoolkit.org website locally to preview your translation. Here is how translated page looks like:

$this->add('HtmlElement')->setElement('A')->setAttr('href',$this->api->getDestinationURL('learn/install/lamp',array('lang'=>'pl')))->set('Polish translation of installation page');
