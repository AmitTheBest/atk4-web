With Agile Toolkit you can be the most efficient and web developer on the block. Here is what you need to learn:

1. Agile Toolkit introduces and follows fundamental concepts. Learn them first.

2. Start new project or download existing example to experement with.

3. Follow a tutorial or explore documentation below:

Application and Framework
Execution steps of typical application; routing; config; logging. Various API classes.
Views, Templates, HTML
Visual elements on your pages. Custom HTML snippets. Introduction to Templates.
Models and ORM (SQL)
Object-Oriented interface to a relational database data. Use directly or with Views.
Form
The fundamental data-entry view. AJAX in Form. Field definitions and control.
Lister
Listing data with user-defined HTML template. Reloading with AJAX; Pagination; Filtering.
Grid
A tabular element for interacting with your data. Standard and custom column types.
CRUD
A swiss-army knife of data editing. Adding, Editing, Deleting in a one-liner.
Menu
A standard widget for displaying user-menu. Look customization, multiple levels.
Authentication
Restricting access to your web application. Multiple user roles. Custom Login Forms.
Testing
Building unit-tests and stress-tests for your back-end. Best practices and extending.
UI Customization
Changing look and feel of your web application entirely. Changing HTML/CSS.
DSQL
Dynamic SQL Abstraction library allows you to build or customize SQL queries.
SMLite
Lightweight template parser of Agile Toolkit. API. Cloning Regions and manipulation.
TMail
Generate emails based on templates. HTML and/or Text. Send using various transports.
Hierarchy
Presenting hierarchical data as trees or drop-downs. Editing tree structures.
noSQL Models
Building support for non-standard database systems. MongoDB, MemCache, Redis.
JavaScript
Integration with jQuery, action chains, event binding. Built-in jQuery widgets.
Third-party integration
jQuery Plugins, Zend, Joomla, Wordpress and CodeIgniter integration.
Filestore
Agile Approach to File Uploads and storage. Image upload. Croping. Thumbnails.
