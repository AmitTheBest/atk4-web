Web Browser

You are on your own. As developer you would need to wrap your code into HTML templates and think about AJAX interractions. This tripples the cost of your software, even if you are experienced developer.

Application Server

Internet is full with PHP-resources, which is why PHP is a #1 web programming language in the world. Most of the stuff is open-source and without commercial backing has very slow growth, low quality and contains large amount of unnecessary features.

Database

You are on your own. Buy that SQL book and start thinking about SQL Injection prevention. You can use 3rd party ORM library which will not work directly with your UI, so even more code for you to write.

Web Browser

You are on your own with most of existing frameworks. They define no standard and any 3rd party plugins you might find would require modifications and tuning to fit into your interface. Code duplication and unnecessary complexity drives your software speed down as a result the only choice left is to spend time on your own UI implementation.

Application Server

Minimalistic frameworks only provide you with fraction of functionality and no structure for your application. Other frameworks are over-complicating things making you write 3-times the code advocated by evangelists as "a good thing".

Database

Frameworks approach database objects very seriously. Some are offer much richer functionality than Agile Toolkit, which generally comes at expense of application speed.

Web Browser

Agile Toolkit is capable of producing HTML templates, JavaScript, jQuery, CSS and AJAX bindings for you. All the views you create or any addon you use will contain a great UI.

Application Server

You write your business code here, not scaterred across multiple places. Any code is object-oriented and scalable. Easy enough to understand for beginners and suitable for high-volume projects.

Database

Similar to other PHP Frameworks, Agile Toolkit supports many concepts such as MVC, ORM, DynamicQueries. Agile Toolkit favours lightweight, extensible and speed-oriented approach.
