jQuery UI Based API and AJAX

Agile Toolkit is based on industry-standard jQuery and jQuery UI to build interactivity between user interface and server-side. Through abstraction, you will be able to harness power of AJAX, jQuery Plugins and jQuery UI Widgets from the comfort of the PHP. Even if your plugin have never been used with Agile Toolkit, integrating it will be a super-easy. Next 2 examples contain zero JavaScript code.
