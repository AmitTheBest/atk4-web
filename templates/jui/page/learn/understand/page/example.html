Practical Example

About Page

Create file templates/default/page/about.html and put some information inside about your application. You can use any HTML markup, but avoid using JavaScript just yet. The content of this page will be inserted into your Shared template.

Registration Page

We will also need to have a separate page for user registration. Create page/register.php.

To make things look nicer, let's use a custom template for registration. Define registratinon page like this:

class page_register extends Page { function init(){ parent::init(); $form = $this->add('Form'); $form->addField('line','email'); $form->addField('password','password'); $form->addSubmit(); } }

As you can see I have created new page object, re-defined init() method and added form into the page. I also have added 2 fields and a submit button into the form.

To customize the look of the registration page, I'd like to redefine the template by adding the following method:

function defaultTemplate(){ return array('page/register'); }

and adding the following template file into templates/default/page/register.html:

<div id="<?_name?>" class="atk-row">
  <div class="span8">
    <h1>Registration form</h1>
    <?$Content?>
  </div><div class="span4">
    <h4>Why Register?</h4>
    <p>You need to apply for new account to be able to access our video rental catalogue</p>
  </div>
</div>

After you've done this, find where Menu is added inside your Application class (lib/Frontend.php) and add new page to it:

$menu->addMenuItem('register');

You should now have a new page with non-functioning form inside a custom markup. Keep this example and we will continue enhancing it in the future.
