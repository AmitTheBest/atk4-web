Introduction to Chains

JavaScript chain is, like everything else, an object in Agile Toolkit. It is a sequence of actions such as traversing or modifications and is very well known to you if you ever used jQuery.

selector.find('div').show('slow');

In Agile Toolkit the same code looks like this:

$view->js(true)->find('div')->show('slow');

Selectors

In jQuery, you need to use $('selector') to specify where to star off. Agile Toolkit automatically starts with the $view of your choice. Any View object have a js() function which will create a new chain and use that object's selector. To add JS code for hiding any object on your page, such as grid, all you need to write is:

$grid->js(true)->hide();

Arguments

When Chains are converted into native JavaScript, arguments are are recursively converted:

$x = 2+2; $js = $page->js(true) ->css(array('width'=>$x)) ->html('O\'Relly'); // Produces: // $('#AgileWeb_learn_understand_chains_intro_doc_example_view') // .css({'width':4}).html('\x3cb\x3eO\x27Relly\x3c/b\x3e')
