Where to find demos and examples?

As mentioned previously, many examples will appear on documentation and introduction pages. Additionally you can find examples on demo.atk4.com. Another location for examples is Examples page on agiletoolkit.org.

Codepad

There is a codepad.agiletoolkit.org which is used by the core team to link to some samples exactly. Codepad is available from Git and you can install it on your own server and share links with others. Sample page is: http://codepad.agiletoolkit.org/issue12?b=devel. If you omit "?b=devel" option, 4.0 version of Agile Toolkit will be used instead of 4.1.
