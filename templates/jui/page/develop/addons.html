Currently available add-ons

Realex payment add-on

If you wish to receive credit/debit card payments - this is all-in-one solution for it. Just add it to your software, enter your Realex Merchant ID and password. It will take care of charging your customers monthly and allow you to amend and refund payments

User statistics/audit add-on

This is the perfect add-on if you wish to see how your users have performed over period of time.

Permission add-on

Granting privileges to one user and denying them to other is always a hassle. This is basic permission module that can be implemented in your application for seamless permission control.

Agile Toolkit CMS

Content Management Systems (CMS) are important for most of the websites. If you want simply to use and great CMS in your application, this is the solution.

Referral add-on

If you wish to award other websites / people who referred client to you, this software is for you

Comments module

User interaction is what makes any website great. Comments module allows user to comment on different pages and for you to administrate them from the admin system.

Tree design

While its not very common, a lot of data can become rather difficult without a tree display. This is a simple add-on that will allow you to move items in tree design, edit, add and delete them.

Parse / import csv files

Another data add-on that will help you take care of seamless import / export of comma separated files (csv) in your database.

Multi-currency

If you are have more than one currency then managing it can sometimes become hard. Use this add-on to simplify it.

Country selection

While there is a lot of country selection add-ons out there, it is hard to find one that fits neatly into your software. Download this simple add-on and amend it for your solution.

Photo widget

Great solution for photo galleries displayed on your website

Video widget

Allows you to display various youtube videos and a video gallery on your website

GEOIP localisation addon

If you need to pinpoint people, offices, cars, shops or more on a map, this is the best GeoIP out there.

User module

This is a simple module allowing you to implement login, registration, user profile and administration right away.
