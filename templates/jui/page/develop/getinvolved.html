If your company produces Web software, consider using Agile Toolkit. If your business environment has some requirements which Agile Toolkit can't meet, please let us know

Promoting Affero GPL Software

Affero GPL v3 is a license which makes sure that any software built on it remains open-source. Unlike other web frameworks which allow closed-source development, we are willing more and more software being built on ATK4 and have it's source code released. If your company needs to build some internal software, you would not loose anything if it would be open-sourced.

Consider about licensing your project with AGPL. Even if it's a private venture, it makes a lot of sense to release open-source version for your product promotion.
