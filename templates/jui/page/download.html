Agile Toolkit 4.0.1

Agile toolkit 4.0.1 is the first stable release of 4.0 branch. You do not need any other frameworks to use Agile Toolkit. Simply download the sample-project or sample-website package and start editing.

Packages

  • atk4-sample-project (Download ZIP) — Download, unpack into the webroot and open in your browser. Sample project comes with both atk4 and atk4-addons and includes heavily documented files which will help you understand the structure of the project and get started. Recommended to new users
  • atk4-sample-website (Download ZIP) — A really simple website using atk4. Download this if you are a designer or are just starting with PHP. This package starts with the basic stuff.
  • atk4 (Download ZIP) — A stand-alone installation of Agile Toolkit core. Extract into your project directory. You can always replace your atk4 directory with more recent version.
  • atk4-addons (Download ZIP) — A stand-alone package containing some of the contributed addons for Agile Toolkit.

Git Repository

  • master (Download ZIP or browse) — this branch contains the most recent stable branch of Agile Toolkit.

  • develop (Download ZIP or browse) — development branch of Agile Toolkit. This repository should be safe to use, although some of the features might not be documented.
