render ( )

Default rendering method. Generates HTML presentation of $this view. For most views, rendering the $this->template would be sufficient.

If your view requires to do some heavy-duty work, please be sure to do it inside render() method. This way would save some performance in cases when your object is not being rendered.

render method relies on method output(), which appeends HTML chunks to the parent's template.
