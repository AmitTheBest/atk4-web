Detailed Description

This is a Firebird/Interbase driver for Dynamic SQL. To be able to use it in your proyects make sure you have the Firebird PDO driver installed for PHP. For more info see PHP manual: http://php.net/manual/en/ref.pdo-firebird.php Howto for compiling/installing on Linux: http://mapopa.blogspot.com/2009/04/php5-and-firebird-pdo-on-ubuntu-hardy.html
